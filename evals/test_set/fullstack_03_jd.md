# Senior Full-Stack Engineer — Retool

**Company:** Retool
**Location:** San Francisco or Remote (US)

## About Retool

Retool is how internal tools get built. We give developers Lego pieces for UI, data, and logic so they can build the apps their company actually needs in an afternoon instead of a quarter.

## What you'll do

- Work across the stack on the Retool Workflows product — the visual cron + event-driven automation surface
- Design and ship the workflow execution engine in TypeScript/Node
- Build the React frontend for the workflow canvas (drag-and-drop, live debugging, history view)
- Own data model decisions in Postgres; we run a single primary with read replicas

## Requirements

- 4+ years writing TypeScript or JavaScript in production
- Hands-on experience with at least one production-grade visual editor or graph UI (you've fought with React-Flow, tldraw, or similar)
- Comfort designing Postgres schemas and writing the migrations to roll them out safely
- Experience with background job systems (BullMQ, Sidekiq, Temporal, our own — any of them)

## Nice to have

- You've shipped a workflow / automation / iPaaS product before
- Hooked up an OAuth integration to a third-party API and lived to tell the tale
