# Senior Full-Stack Engineer — Cron / Notion Calendar

**Company:** Notion
**Location:** New York or Remote (US/Canada)
**Team:** Calendar (formerly Cron, acquired 2022)

## About the team

Notion Calendar is the keyboard-driven calendar that thousands of operators, founders, and engineers swear by. We're a small team — fewer than 15 engineers — and we ship continuously.

## What you'll do

- Build features across our Electron + React desktop client and our Node/TypeScript backend
- Own a slice of the calendar sync engine that talks to Google Calendar, Microsoft 365, and iCloud
- Investigate weird bugs in iCal RFC 5545 (you will learn more about recurrence rules than you ever wanted)
- Improve performance — the desktop app holds itself to a hard 60fps target

## Requirements

- 5+ years shipping TypeScript / React in production
- You've shipped a desktop app via Electron, Tauri, or native — or a mobile app and you know what an event loop being blocked looks like
- Comfort with offline-first sync conflict resolution
- You think the iCal spec is a charming historical document, not an insult

## Compensation

- $190-240k base + equity, depending on level and location
