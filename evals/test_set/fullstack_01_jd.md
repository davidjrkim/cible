# Senior Full-Stack Engineer — Linear

**Company:** Linear
**Location:** Remote (Europe / Americas timezones)
**Team:** Product Engineering

## About Linear

Linear is the issue tracking tool used by thousands of high-performing software teams. We're building the tool we wished we had — fast, keyboard-driven, opinionated, and beautiful. Our customers ship faster because Linear gets out of their way.

## What you'll do

- Own end-to-end product features across our TypeScript stack (Next.js, React, GraphQL, Postgres)
- Ship code that is read by your peers within hours, not weeks
- Work directly with designers and PMs in 4-person product pods
- Improve our offline-first sync engine that keeps Linear feeling instant
- Contribute to our public API and SDK used by integrators like Vercel, Plain, and Cron

## What we look for

- 5+ years building production web applications
- Deep TypeScript and React expertise
- Comfort with Postgres, including writing non-trivial SQL and reasoning about query plans
- Strong product instincts — you push back on bad specs
- Bias to shipping. We deploy multiple times per day; long-running branches are a smell.
- Bonus: experience with real-time sync, CRDTs, or local-first software

## What we don't expect

- A computer science degree
- Familiarity with our specific stack on day one — we'll pair you with someone for the first two weeks

## Compensation

- Competitive salary + equity
- Fully remote, async-first
- 32 days PTO + your country's public holidays
