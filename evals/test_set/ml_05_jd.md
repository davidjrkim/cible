# ML Engineer — Cresta (Voice AI)

**Company:** Cresta
**Location:** Remote (US)
**Team:** Speech / NLP

## About the role

Cresta builds real-time AI agents that coach human contact-center agents during live calls. We need ML engineers comfortable across speech recognition, fine-tuning small LMs, and shipping low-latency inference to production.

## What you'd do

- Fine-tune small (1-8B parameter) language models for our domain-specific scoring + summarization tasks
- Improve our streaming ASR pipeline (latency, accuracy on customer-specific vocabularies)
- Build evals for production agent quality — these are real-time, so eval design is harder than for batch tasks
- Work directly with deployed customer accounts; data is messy and the eval distribution is shifting

## Requirements

- 3+ years applied ML in production
- Experience fine-tuning LMs (LoRA / QLoRA / full SFT) on domain data
- Comfort with ASR — Whisper, Conformer, Distil-Whisper, anything
- You've built an eval harness for a production system, not just for a research paper

## Bonus

- Real-time / streaming inference experience
- You've owned a customer-facing ML feature end-to-end (data pipeline → eval → deploy → on-call)
