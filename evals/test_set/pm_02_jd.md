# Senior Product Manager — Vercel (AI SDK)

**Company:** Vercel
**Location:** Remote (US / Europe)
**Team:** AI SDK

## About the role

You'd be the PM for Vercel's open-source AI SDK — the TypeScript library used by tens of thousands of developers to build LLM-powered applications. The PM role is half product strategy, half developer-experience research.

## What you'd do

- Own the AI SDK roadmap. Decide which model-provider features to support, what abstractions to expose, what to deprecate
- Talk to developer customers weekly — both at our enterprise accounts and from the open-source community
- Write public RFCs for major API changes. Discussion happens on GitHub.
- Partner with two engineers and a designer on a small pod

## Requirements

- 4+ years product management, ideally in developer tools
- Technical depth — you can read TypeScript, run an example end-to-end, and debug why it broke
- You've shipped an SDK, library, or API as a product (not just used one)
- Strong written communication — RFCs, docs, changelogs

## Bonus

- You've built something on top of Anthropic, OpenAI, or another LLM API
- You've maintained a meaningful open-source project
