# Full-Stack Engineer — Vercel

**Company:** Vercel
**Location:** Remote (US / Europe)
**Team:** Dashboard

## About the role

The Vercel dashboard is the surface most developers see every day. We're a small product team responsible for project settings, deployments, observability, and billing UIs. We move fast and ship behind feature flags.

## What you'll do

- Build features end-to-end in Next.js (App Router), React Server Components, and tRPC
- Own UX flows from spec to launch, including writing the migrations and instrumenting metrics
- Improve the dashboard's performance — we hold ourselves to Core Web Vitals "good" on every route
- Contribute to our internal design system (built on Radix + Tailwind)

## What we look for

- 3+ years of production React work, ideally with the App Router or Remix
- Familiarity with edge runtimes and streaming SSR
- Experience instrumenting your own features (we use OpenTelemetry + Datadog)
- Good taste — you notice when a loading spinner is in the wrong place

## Bonus

- Open source contributions to Next.js, SWR, or related libraries
- Experience working on a developer-tools product
