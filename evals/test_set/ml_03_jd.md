# Applied ML Engineer — Anthropic

**Company:** Anthropic
**Location:** San Francisco or London
**Team:** Applied AI

## About the team

Applied AI builds the production AI features customers use on top of Claude — tool use, agentic loops, document-aware retrieval, structured outputs. We sit between research and product.

## What you'd do

- Build the eval harnesses that gate any change to a production prompt, fine-tune, or system prompt
- Design new agentic capabilities (tool use, structured outputs, computer use) and prove they work via held-out evals before shipping
- Iterate on prompting + retrieval pipelines for customer-facing features
- Write internal post-mortems when a capability regresses in production

## Requirements

- 3+ years applied ML or ML infrastructure
- Hands-on experience writing eval harnesses for LLM-driven systems — not "I read about evals," "I have shipped one that gated a deploy"
- Comfort with the prompt-iteration loop: hypothesis → eval → measure → ship or revert
- Strong Python; comfort with at least one production language alongside (TypeScript / Go / Rust)

## Bonus

- You've shipped a feature using tool use, retrieval, or computer use in production
- You think cross-family judges are important (we do)
