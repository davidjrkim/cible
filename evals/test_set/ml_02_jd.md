# Senior ML Engineer — Hugging Face

**Company:** Hugging Face
**Location:** Remote (any timezone)
**Team:** Transformers + Accelerate

## About the role

You'd work on Hugging Face's core OSS — `transformers` and `accelerate`. Used by hundreds of thousands of researchers and engineers worldwide.

## What you'd do

- Implement new model architectures in PyTorch as the ML literature ships them (multimodal, MoE, speech, you name it)
- Improve distributed-training UX in `accelerate` — FSDP, DeepSpeed-style ZeRO, tensor parallelism
- Optimize inference paths (kv-cache management, paged attention, speculative decoding)
- Triage and merge community PRs

## Requirements

- 4+ years writing production PyTorch
- You've implemented a transformer model from a paper, end to end
- Comfortable with CUDA at the level of "I've written a Triton kernel" or "I've debugged a CUDA OOM that wasn't obvious"
- You care about library ergonomics — `transformers` ships to a huge audience

## Bonus

- You've contributed to `transformers`, `accelerate`, `peft`, or similar
- Multi-GPU distributed training experience (FSDP, DeepSpeed, Megatron)
