# Senior Full-Stack Engineer — Raycast

**Company:** Raycast
**Location:** Remote (Europe)
**About:** Raycast is the productivity launcher for macOS — Spotlight, but useful.

## The role

We're hiring a senior full-stack engineer to work on the Raycast cloud surface: account management, the extension store, AI features, and team plan administration. You'll write Swift occasionally but mostly TypeScript.

## You will

- Own backend services in Node/TypeScript on AWS (Lambda + Postgres + DynamoDB)
- Build the web dashboard (Next.js, Tailwind) for managing teams, billing, and extensions
- Ship the extension marketplace surfaces — discovery, reviews, and the safety review pipeline
- Investigate and fix tricky Mac client ↔ cloud sync bugs (you'll touch Swift, but you don't need to be an expert)

## Requirements

- 5+ years writing TypeScript in production
- Comfortable on AWS — at least Lambda + Postgres + S3 + IAM
- You've worked on a billing or marketplace product
- Strong product taste (this is Raycast — taste is the bar)

## Nice to have

- You've written Swift, even casually
- You've shipped a Raycast extension (we will absolutely look)
