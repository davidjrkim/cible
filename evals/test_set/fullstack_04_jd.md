# Full-Stack Engineer — PostHog

**Company:** PostHog
**Location:** Fully remote
**Compensation:** Public salary bands on our website

## About PostHog

PostHog is the open-source product analytics platform. 70k+ companies use us, and our entire codebase is on GitHub. Everything we build is in the open.

## What you'll do

- Work on the Insights product surface — funnels, retention, paths, SQL editor
- Ship in our Django + Python backend and our React + TypeScript frontend
- Talk to users directly (we have no PMs; engineers are responsible for talking to customers)
- Write a public-changelog entry for every shipped feature

## What we look for

- 3+ years building production web apps
- Comfortable in both Python and TypeScript — we hire generalists
- Experience with at least one analytical database (ClickHouse, Pinot, BigQuery, Snowflake)
- Strong product sense and a habit of using your own product

## Bonus

- You've built BI or analytics tooling before
- You've written SQL by hand for a week and not switched to an ORM at the end of it
