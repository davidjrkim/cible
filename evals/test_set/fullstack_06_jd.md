# Full-Stack Engineer — Plain

**Company:** Plain
**Location:** Remote (Europe only)
**Funding:** Series A, ~$15M raised

## What is Plain?

Plain is a B2B customer support platform built for software teams. We're the support tool you actually want to use — fast, keyboard-first, well-typed, and tightly integrated with your existing stack.

## What you'll work on

- Our customer-facing inbox app (Next.js + GraphQL + Postgres)
- The webhooks + integration framework that lets customers route Slack, Discord, and Linear events into Plain
- Performance work: we hold ourselves to <100ms p95 server response on the inbox
- Public API and docs

## What we want

- 3-6 years of production TypeScript
- You've built things on top of GraphQL (we use Apollo Server + codegen)
- You enjoy writing types more than you enjoy avoiding them
- You can write at least a paragraph about caching in 60 seconds without making things up

## What we don't care about

- Your CS degree
- Whether you've used "our exact" stack before
