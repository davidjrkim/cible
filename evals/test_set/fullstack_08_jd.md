# Full-Stack Engineer — Resend

**Company:** Resend
**Location:** Remote (any timezone)

## What is Resend?

Resend is the email API for developers. Think Stripe for email. We're a small team (under 30) and we ship fast.

## What you'll do

- Build the Resend dashboard (Next.js + tRPC + Postgres)
- Own the email-event ingestion pipeline (we process ~50M events/day across SES, our own MTA, and partner ESPs)
- Build features for our open-source React Email package
- Talk directly to developer customers — no PM layer between you and them

## What we look for

- 3+ years TypeScript production experience
- You've shipped product features at a developer-tools company (or it's clear you've used enough developer tools to have opinions)
- Comfort with email — SPF, DKIM, DMARC, bounces, suppressions, the whole horrible mess
- You can write a CHANGELOG entry that a developer will actually read

## Bonus

- You've contributed to React Email, Nodemailer, or postal-mime
- You've operated an MTA (Postfix, Haraka, our own) in production
