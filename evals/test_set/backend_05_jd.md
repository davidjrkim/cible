# Senior Backend Engineer — Temporal

**Company:** Temporal Technologies
**Location:** Remote (US/Europe)

## About Temporal

Temporal is the durable execution platform — we run workflows that survive process restarts, machine failures, and data center outages. Used by Snap, Stripe, HashiCorp, and thousands of others.

## What you'd do

- Work on the Temporal server (mostly Go) — the matching, history, and frontend services
- Improve sharding behavior under load: we run 4k+ shards per customer cluster and the rebalancing logic has rough edges
- Investigate gnarly distributed-systems bugs (split-brain, replication lag, the works)
- Contribute to public OSS — Temporal is Apache-2 licensed and the work is in the open

## Requirements

- 5+ years backend experience
- Strong Go
- You've worked on a distributed system at scale (Cassandra, Kafka, etcd, ScyllaDB, Temporal itself, anything in this family)
- You can read Lamport, Raft, and Paxos papers without your eyes glazing over

## Bonus

- You've operated a Temporal cluster in production
- You've contributed to Cadence, Temporal, or another durable-execution system
