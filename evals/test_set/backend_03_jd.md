# Senior Backend Engineer — Clickhouse Cloud

**Company:** ClickHouse Inc.
**Location:** Remote (US/Europe)
**Team:** Cloud Platform

## What you'd work on

ClickHouse Cloud is the managed-service version of the open-source OLAP database. You'd work on the control plane that provisions, scales, and bills customer clusters across AWS, GCP, and Azure.

## Responsibilities

- Build and operate the Kubernetes-based control plane (Go, Crossplane, custom operators)
- Own multi-cloud provisioning: customer clicks "create cluster" → cluster is healthy in <90 seconds
- Improve our autoscaler that resizes compute based on query load
- Carry an on-call pager. ClickHouse Cloud has a public SLA and we mean it.

## Requirements

- 5+ years backend engineering in production environments
- Hands-on Kubernetes-operator experience (you've written one, not just deployed Argo)
- Strong Go
- You've supported a multi-cloud product or at least taken AWS + one other to production

## Bonus

- Database internals knowledge — you've debugged a query planner before
- Familiar with Crossplane, OpenTelemetry, or Prometheus operators
